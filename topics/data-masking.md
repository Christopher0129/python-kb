# 数据脱敏专题

这个专题关注如何在开发、测试、分析和日志场景下减少敏感数据暴露。

## 适用场景

- 生产数据回流测试环境
- 日志和审计记录用户字段
- 数据分析和报表需要弱化可识别信息

## 常见主题

- 静态脱敏
- 动态脱敏
- Tokenization
- 哈希与不可逆处理
- 角色视图隔离

## 设计要点

- 不同场景目标不同：测试重可用性，日志重最小暴露，分析重统计保真
- 脱敏策略要与字段分类体系配合
- 不要把“部分打码”误当作充分保护
- 对导出、回放和排障链路也要有一致策略
