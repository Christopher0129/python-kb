# Docker 专题

这个专题帮助 Python 项目建立最基础的容器化认知：镜像、容器、依赖封装和运行入口。

## 核心价值

- 本地与线上环境更一致
- 部署更可重复
- 依赖打包更清晰

## 常见要点

- 选择基础镜像
- 安装依赖
- 复制项目代码
- 指定启动命令
