# Linux 与 Python 运维专题

这个专题关注 Python 服务在线上环境的基础运维视角：进程、日志、权限、端口和目录。

## 常见排查点

- 进程是否在运行
- 日志是否有异常堆栈
- 端口是否监听
- 文件权限是否正确
- 配置是否加载到了预期值

## 常见能力

- 用 `subprocess` 调系统命令
- 用 `logging` 记录排查信息
- 用 `pathlib` 检查目录和文件
