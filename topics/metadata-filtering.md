# 元数据过滤专题

这个专题关注在全文检索、向量检索和混合检索中如何使用结构化字段做过滤与分桶。

## 适用场景

- 文档需要按租户、语言、版本或时间过滤
- 搜索结果需要限定在某个产品线、知识域或权限范围内
- 需要减少跨集合误召回

## 常见主题

- 预过滤与后过滤
- ACL / 租户过滤
- 时间范围过滤
- 标签与分类过滤
- Facet 与聚合统计

## 设计要点

- 过滤字段要在索引设计时前置规划，不能事后硬补。
- 强约束过滤尽量前置，避免召回后再大规模丢弃。
- 过滤条件越复杂，越要关注搜索引擎和向量库的执行模型差异。
- 权限过滤必须可证明正确，不能只依赖前端约束。
