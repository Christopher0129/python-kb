# OAuth2 与 OpenID Connect 专题

这个专题关注现代身份系统里的授权委托和身份声明，而不是只做本地登录。

## 概念区分

| 概念 | 说明 |
| --- | --- |
| OAuth2 | 授权框架 |
| OIDC | 建立在 OAuth2 上的身份层 |
| Access Token | 访问资源用 |
| ID Token | 表示身份信息 |

## 适用场景

- 第三方登录
- 统一身份平台接入
- 服务间标准化认证
