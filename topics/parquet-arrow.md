# Parquet 与 Arrow 专题

这个专题关注列式文件格式和内存列式表示在 Python 数据工程中的作用。

## 核心概念

| 概念 | 说明 |
| --- | --- |
| Arrow | 内存列式格式 |
| Parquet | 面向分析的列式存储格式 |

## 常见价值

- 高效读取部分列
- 更适合分析任务
- 与 DuckDB、Polars 等工具互通
