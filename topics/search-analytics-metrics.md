# 搜索分析指标专题

这个专题关注搜索系统除了延迟和 QPS 之外，还该长期追踪哪些效果与行为指标。

## 适用场景

- 团队只看成功率和 RT，却很难解释“效果变差”
- 需要建立搜索实验和日常监控的共同指标面板
- 不同查询类型的表现差异很大

## 常见主题

- 零结果率
- reformulation rate
- CTR
- query coverage
- 结果位置分布

## 设计要点

- 搜索分析要按查询意图、租户、过滤场景和终端类型分层看。
- 只看全局均值很容易掩盖头部查询和长尾查询的不同问题。
- 行为指标需要和离线标注、线上实验和投诉样本一起解释。
- 指标体系应区分召回问题、排序问题、展示问题和内容供给问题。
