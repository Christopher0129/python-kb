# 测试进阶专题

这个专题关注比基础断言更进一步的测试能力：fixture 设计、分层策略、性能测试和外部依赖隔离。

## 核心主题

- fixture 复用边界
- mock 的使用粒度
- 接口测试与集成测试区分
- 性能测试和回归基准
- 契约测试与 schema 校验
